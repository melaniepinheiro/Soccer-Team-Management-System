## Project Description

This project is a Soccer Team Management System created in Java. The program allows users to work with different types of soccer teams, including club teams and national teams. Users can calculate team revenue, update team rosters, and view information about each team. The main goal of this project was to practice object-oriented programming concepts such as inheritance, polymorphism, interfaces, and abstract classes.

## Programming Concepts Used

* Interfaces
* Abstract Classes
* Inheritance
* Polymorphism
* Arrays
* Constructors
* Method Overriding
* User Input with Scanner

## Inputs

The user enters information such as:

* Ticket prices
* Number of home games
* Merchandise revenue
* Sponsorship amounts
* Number of tournaments
* Roster changes
* Coach names

## Outputs

The program displays:

* Team information
* Team type
* Estimated season revenue
* Updated roster information
* Updated budget information

## What I Learned

This project helped me better understand how inheritance and polymorphism work in Java. It also gave me more experience creating classes, using interfaces, and organizing a program with multiple files.
